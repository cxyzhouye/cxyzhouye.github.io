Linux系统目录结构

​ /dev :存放抽象硬件
​ /lib :存放系统库文件
​ /sbin :存放特权级二进制文件
​ /var :存放经常变化的文件
​ /home :普通用户目录
​ /etc :存放配置文件目录
​ /boot :存放内核与启动文件
​ /bin :存放二进制文件(可执行命令)
​ /usr :存放安装程序(软件默认目录)
​ /mnt :文件挂载目录(U盘,光驱)
​ /root :特权用户目录
​ /opt :大型软件存放目录
​ /tmp :存放临时文件

Linux常用命令

​ Linux是一个多用户,多任务的操作系统
​ 用户组 :就是具有相同特则的用户(user)的集合体；(我们可通过定义用户组和修改文件的权限来实现)
​ 用户和用户组的关系 :
​ 一对一 :一个用户可以存在一个组里，是组中的唯一成员；
​ 一对多 :一个用户可以存在多个组里，此用户具有这多个组的共同权限；
​ 多对一 :多个用户存放一个组中，这些用户具有和组相同的权限；
​ 多对多 :多个用户可以存在多个组中，是以上3中关系的扩展。

1.用户和用户组命令：

​ root :UID是0
​ 系统用户的UID为1~999
​ 普通用户的UID从1000开始

添加用户:

​ useradd(选项)(参数)
​ -d 指定用户的家目录(默认为/home/username)
​ -e 用户到期时间,格式为YYYY-MM-DD.
​ -u 指定该用户的默认UID
​ -g 指定一个初始的用户基本组
​ -s 指定该用户默认Shell解释器

修改用户：

​ usermod(选项)(参数)
​ -c<备注> :修改用户账号的备注文字
​ -d<登入目录> :修改用户登入时的目录
​ -e<有效期限> :修改账号的有效期限
​ -f<缓冲天数> :修改在密码过期后多少天既关闭该账号
​ -g<群组> :修改用户的群组
​ -u :修改UID
​ -U :解除密码锁定

用户账号口令管理:(用户修改用户密码，过期时间，认证信息等)

​ userwd(选项)(参数)
​ -d :删除密码，仅有系统管理者才能使用
​ -f :强制执行
​ -k :设置只有在密码过期失效后，方能更新
​ -l :锁住密码
​ -s :列出密码的相关信息，仅有系统管理者才能使用
​ -u :解开已上锁的账号

删除用户:

​ userdel -rf (用户名)

添加用户组：

​ groupadd [选项] [用户组名]
​ -g 指定新用户组的组ID（GID）。
​ -o 一般与-g选项同时使用，表示新用户组的GID可以与系统已有用户组的GID相同。
​ 范例：
​ groupadd -g 101 group2 #创建一个组ID为101的组

删除用户组

​ groupdel
​ 范例：
​ groupdel group2

修改用户组

​ groupmod [选项] [用户组]
​ -g GID 为用户组指定新的组标识号。
​ -o 与-g选项同时使用，用户组的新GID可以与系统已有用户组的GID相同。
​ -n新用户组 将用户组的名字改为新名字
​ 范例：groupmod –g 10000 -n group3 group2 #将组group2的标识号改为10000，组名修改为group3。
​ Linux下用户分为三类：超级用户，普通用户，伪用户

用户切换
用户身份切换

​ su
​ sudo [参数] 命令名称
​ -h 列出帮助信息
​ -l 列出当前用户可执行的命令
​ -u 用户名或UID以指定的用户身份执行命令
​ -k 清空密码的有效时间，下次执行sudo时需要再次进行密码验证
​ -b 在后台执行指定的命令
​ -p 更改询问密码的提示语
​ 编辑配置文件命令 :visudo

文件基本属性 :
文件类型 :

​ 当为d则是目录
​ 当为 - 则是文件；
​ 若是 l 则表示为链接文档(link file)；
​ 若是 p 则表示为装置文件里面的可供储存的接口设备(可随机存取装置)，管道文件
​ 若是 c 则表示为装置文件里面的串行端口设备，例如键盘、鼠标(一次性读取装置)

文件权限 :

​ 读(r) -允许读取文件
​ 写(w) -允许写，修改和删除某个文件
​ 执行(x) -运行执行某个文件
​ 改变所属群组 : chgrp
​ chgrp [-R] 属组名 文件名 -R :递归更改文件属性，就是在更改某个目录文件的属组时，如果加上-R 的参数，那么该目录下的所有文件的属组都会更改

改变文件属性 :chmod

​ 一、用u、g、o 来代表三种身份的权限 a 则代表 all
​ 二、r:4，w:2，x:1

改变文件拥有者 :chown

​ 更改文件属主，也同时更改文件属组
​ chown [-R] 属主名 文件名
​ chown [-R] 属主名 : 属组名 文件名
